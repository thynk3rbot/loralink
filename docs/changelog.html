Changelog

Version history and release notes for LoRaLink-AnyToAny.

v1.5 — March 2026

  • MQTT integration with Swarm Dashboard
  • Unified startup script for development environment
  • Node registry with clear-all functionality
  • BLE scan auto-refresh in web UI
  • Toast notification system for command feedback
  • Navigation links between dashboard and Swarm view
  • Safari CSS compatibility fixes

v1.4 — February 2026

  • Web dashboard redesign with SCADA-density layout
  • Pin scheduling GUI with categorized interface
  • Persistent pin names across reboots
  • Config export/import with hardware ID validation
  • Golden image backup and restore
  • Network discovery (subnet scan)
  • Protocol verification and analysis tools

v1.3 — January 2026

  • ESP-NOW support with peer management
  • BLE GATT terminal interface
  • ACK + retry system for reliable delivery
  • Dynamic task scheduling via JSON

v1.2 — December 2025

  • AES-128 encrypted LoRa communication
  • Web dashboard with live status monitoring
  • REST API for remote control
  • OTA firmware updates

v1.0 — November 2025

  • Initial release
  • LoRa mesh communication
  • Serial command interface
  • Basic GPIO control
