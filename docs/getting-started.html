Getting Started

Go from zero to a working LoRaLink mesh in under 10 minutes.

Prerequisites

  • PlatformIO installed (VS Code extension recommended)
  • Heltec WiFi LoRa 32 V3 board(s)
  • USB-C cable

Install & Flash

  1. Clone the repository
    git clone <repo-url>
  2. Build & upload firmware
    pio run -t upload
  3. Open serial monitor
    pio device monitor

First Boot

On first boot you'll see the OLED display the device name, firmware version, and IP address (if WiFi is configured). The serial console will show all initialization steps.

Connect to the Dashboard

Navigate to your device's IP in any browser. The web dashboard provides full control: status cards, protocol badges, message log, and command input.

💡 Tip

If you don't know the device IP, check the serial output or OLED display on boot.

Send Your First Command

From the serial console or web dashboard, try:

LED ON
STATUS
BLINK

See the API Reference for all available commands.
