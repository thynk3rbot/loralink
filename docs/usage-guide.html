📖 Usage Guide

A task-oriented walkthrough of every LoRaLink feature. Each section is self-contained — jump to what you need.

Device Setup

Naming Your Device

Set a friendly name (1-14 characters) that identifies the node on the mesh:

SETNAME FarmNode1

Configuring WiFi

SETWIFI MyNetwork MyPassword
SETIP 192.168.1.100    # Static IP
SETIP OFF              # DHCP

Sending Commands

Commands follow the syntax: [TARGET] COMMAND [ARGS]

# Local commands
LED ON
GPIO 5 1
STATUS

# Targeted commands
FarmNode2 LED ON
FarmNode2 STATUS

# Broadcast to all nodes
ALL STATUS
ALL GPIO 5 0

Relay & GPIO Control

LoRaLink supports 4 GPIO-controlled relay outputs. Relays default to LOW on boot for safety.

RelayGPIO PinVoltage
Relay 15110V AC
Relay 24612V DC
Relay 3612V DC
Relay 4712V DC

Scheduling & Automation

Create scheduled GPIO tasks with Toggle, Pulse, and PWM modes via the web dashboard or JSON commands.

SETSCHED 5000        # Toggle 110V relay every 5 seconds
GETSCHED             # View current schedule as JSON

Web Dashboard

The LoRaLink web UI provides tabbed access to all functionality:

  • Fleet — Live node roster, field monitor, topology graph
  • Hardware — Network configuration, node registry, BLE scanning
  • Admin — Config files, golden image backup, protocol verification
  • Swarm — MQTT dashboard with real-time telemetry and broadcast commands

MQTT & Swarm Dashboard

Connect your nodes to an MQTT broker for cloud telemetry and remote control:

CONFIG SET MQTT_SRV 192.168.1.100
CONFIG SET MQTT_PRT 1883
CONFIG SET MQTT_EN true

The Swarm Dashboard at /static/mqtt_dashboard.html provides a real-time grid of all MQTT-connected nodes.

Configuration Management

  • NVS Persistence — All settings survive power cycles
  • Export/Import — JSON-based config backup and restore
  • Factory Reset — WIPECONFIG to erase all settings

OTA Firmware Updates

Push firmware over WiFi using ArduinoOTA. No physical access required once WiFi is configured.

Security

  • AES-128 — All LoRa traffic encrypted with configurable key
  • Sync Word — Private 0x12 isolates your mesh from public LoRa
  • Checksum — Rejects corrupted and noise packets at hardware layer
