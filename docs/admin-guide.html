🛡 Admin Guide

Fleet management, node discovery, deployment profiles, and backup/restore.

For details on managing the background services, Docker, and automatic startup, see the System Operations guide.

Fleet Management

The PC webapp acts as the fleet administrator, providing a centralized view of all nodes with batch control capabilities.

  • Node Registry — Add, connect, and remove devices from the roster
  • Network Discovery — Scan your subnet for LoRaLink devices
  • BLE Scanning — Discover nearby Bluetooth devices and pair peers

Product Profiles & Deployment

Create reusable "product profiles" that define GPIO configurations, schedules, and settings for specific hardware deployments.

Golden Image Backup

Back up a device's complete NVS configuration as a JSON "golden image" for cloning to other units.

# Via the webapp Admin tab:
# 1. Click "Backup Device NVS"
# 2. JSON saved to workstation
# 3. Restore to any compatible hardware

Config Sync

  • Export — GET /api/config pulls full device config
  • Import — POST /api/config/apply pushes config with hardware ID validation

Protocol Verification

The Admin tab includes tools to verify protocol health, trace message routing, and analyze link saturation between peers.
