API Reference

Complete reference for REST endpoints, serial commands, and MQTT topics.

REST API Endpoints

MethodEndpointDescription
GET/api/statusDevice status (ID, battery, IP, RSSI, uptime)
POST/api/cmdSend a command — body: {"cmd":"LED ON"}
GET/api/peersList known mesh peers
GET/api/scheduleGet current GPIO schedule
POST/api/scheduleSet GPIO schedule (JSON body)
GET/api/transportTransport status (BLE, HTTP, peers)
POST/api/rescanTrigger BLE rescan
GET/api/nodesList configured nodes
POST/api/nodesAdd a node
DELETE/api/nodes/{id}Remove a node
POST/api/nodes/clearRemove all nodes
GET/api/ble/discoveredLast BLE scan results
GET/api/discoverStart network subnet scan
GET/api/discover/resultsGet subnet scan results
GET/api/configDevice configuration export
POST/api/config/applyApply configuration JSON

Serial Commands

CommandDescription
LED ON/OFFToggle built-in LED
STATUSDevice status (ID, battery, IP, RSSI)
BLINKFlash LED 10 times
SETNAME <name>Set device name (1-14 chars)
SETWIFI <ssid> <pass>Configure WiFi credentials
SETIP <ip> / OFFStatic IP or DHCP
ESPNOW ON/OFFEnable/disable ESP-NOW
REPEATER ON/OFFEnable/disable LoRa repeater
GPIO <pin> <0/1>Set GPIO pin state
READ <pin>Read GPIO pin
SLEEP <hours>Deep sleep (0.01-24h)
RADIODump LoRa diagnostics
CONFIG SET <key> <val>Set NVS config value
WIPECONFIGFactory reset all settings
<target> <cmd>Send command to specific node
ALL <cmd>Broadcast to all nodes

MQTT Topics

TopicDirectionDescription
loralink/telem/{nodeId}PublishTelemetry data (battery, RSSI, uptime, GPS)
loralink/status/{nodeId}PublishOnline/offline status and metadata
loralink/msg/{nodeId}PublishForwarded mesh messages
loralink/cmd/{nodeId}SubscribeSend command to specific node
loralink/cmd/allSubscribeBroadcast command to all nodes
