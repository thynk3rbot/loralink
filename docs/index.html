What Is LoRaLink?

A production-grade ESP32-S3 firmware framework that makes every wireless interface a first-class citizen — LoRa, ESP-NOW, Bluetooth, WiFi, Serial, and MQTT all route through a single, unified command pipeline.

🔗

Multi-Protocol

LoRa, ESP-NOW, BLE, WiFi, Serial, MQTT — commands route between any interface automatically.

🔒

AES-128 Encrypted

All LoRa traffic encrypted with configurable keys. Private sync word isolates your mesh.

🖥

Web Dashboard

Live status, message log, protocol badges, command input, peer map — all from a browser.

⏱

Task Scheduling

Dynamic GPIO scheduling with Pulse, Toggle, and PWM modes. Configure via JSON or web UI.

📡

MQTT & Cloud

Publish telemetry to any MQTT broker. Integrate with Home Assistant, Node-RED, or custom dashboards.

💰

Sub-$30 Per Node

Built on the Heltec WiFi LoRa 32 V3. Multi-kilometer mesh for the cost of a lunch.
